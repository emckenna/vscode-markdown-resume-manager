# Change Log

All notable changes to the "Markdown Resume Manager" extension will be documented in this file.

## [1.0.0] - 2025-11-03

### Added
- Initial release of Markdown Resume Manager
- Interactive document creation with native VS Code prompts
- Create resumes with company and position prompts
- Create cover letters with company prompts
- Clipboard paste support using VS Code's native API
- One-click building to DOCX format
- Build to both DOCX and PDF formats
- Open build folder directly from VS Code
- Configurable build script path
- Configurable output filenames
- Cross-platform support (Windows, macOS, Linux)
- Keyboard shortcuts for all major operations
- Smart file organization by company name

### Features
- `Resume Manager: Create New Resume` command
- `Resume Manager: Create New Cover Letter` command
- `Resume Manager: Build Current Document (DOCX)` command
- `Resume Manager: Build Current Document (DOCX + PDF)` command
- `Resume Manager: Open Build Folder` command
- Configuration options for customization
- Automatic directory creation
- File overwrite protection
- Build progress notifications
- Error handling and user-friendly messages
